### 1. VERIFICAR VARIABLE
```
Vercel → Settings → Environment Variables
Buscar: OPENAI_TEMPERATURE
```

**¿Qué valor tiene?**
- [ ] Dice "0.7" → Cambiar a `0.0`
- [ ] Dice "0.0" → Ya está bien, solo falta redeploy
- [ ] No existe → Crear con valor `0.0`

---

### 2. CAMBIAR VALOR

**Editar variable existente:**
```
⋮ → Edit → Value: 0.0 → Save
```

**O crear nueva:**
```
Add New
  Name: OPENAI_TEMPERATURE
  Value: 0.0
  Environment: ✅ Production ✅ Preview ✅ Development
  Save
```

---

### 3. REDEPLOY SIN CACHÉ

```

❌ DESMARCAR "Use existing Build Cache"

```
